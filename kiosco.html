            Estructura de la red

            Fundador / Empresa Ferriol OS role: super Aprueba solicitudes · configura comisiones · gestiona la red completa
            Supabase: super
            Alta aprobada por Ferriol · kit inicial pagado
            Administrador de Red Socio / Distribuidor digital role: partner Vende kits y suscripciones mensuales · gestiona sus referidos · recibe comisiones en Ingresos
            Supabase: partner
            Alta vía link de referido · período de prueba
            Negocio (Kiosco) Kiosquero role: kiosquero Usa el sistema en el local · paga suscripción mensual · gestiona productos, caja y deudores
            Supabase: kiosquero

            Flujo de dinero

            Kit inicial vendedor $ 60.000
            80 % → Admin de red (período intro, 1er mes)
            50 % → Admin de red (comisión normal desde mes 2)
            20–50 % → Ferriol (reserva empresa)

            El admin recibe el cobro del cliente y transfiere el 20% a Ferriol (o el split configurado). Ferriol verifica y acredita la comisión en el libro MLM.

            Suscripción mensual negocio (kiosco) $ 9.900 / mes
            80 % → Admin de red (intro, mes 1)
            50 % → Admin de red (normal, mes 2+)
            20 % → Ferriol (siempre)

            El kiosco paga al admin. El admin transfiere el 20% a Ferriol. Ferriol aprueba la solicitud y suma los días al contador del kiosco.

            Cuota mantenimiento admin (mensual) $ 20.000 / mes
            100 % → Ferriol (licencia de distribución)

            El admin paga directamente a Ferriol para mantener activo su rol de distribuidor. Sin esta cuota puede pausarse la cuenta.
